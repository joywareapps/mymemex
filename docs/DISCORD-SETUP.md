# Discord Server Setup Guide

Librarian Community Discord: [discord.gg/YOUR_INVITE_CODE](https://discord.gg/YOUR_INVITE_CODE)

## Channel Structure

### 📋 Information
- `#welcome` — Rules, getting started
- `#announcements` — Releases, updates (locked)
- `#roadmap` — What's coming next (locked)

### 💬 General
- `#general` — Casual chat
- `#introductions` — Say hi, share use cases
- `#show-and-tell` — Your setups, workflows

### 🛠️ Help & Support
- `#installation-help` — Setup issues
- `#configuration` — Config questions
- `#mcp-integration` — Claude Desktop, OpenClaw
- `#feature-requests` — Ideas and suggestions

### 🧪 Development
- `#development` — Code discussion
- `#contributing` — PRs, issues, docs
- `#testing` — Beta testing, RC feedback

### 🎁 Supporters (Restricted)
- `#supporter-lounge` — General supporter chat
- `#early-access` — Beta builds, RC downloads
- `#priority-support` — Faster help for supporters
- `#patron-chat` — Patron-only discussions

## Roles

| Role | Color | Who Gets It |
|------|-------|-------------|
| `@Founder` | Gold | You |
| `@Moderator` | Purple | Trusted community members |
| `@Contributor` | Green | 1+ merged PR |
| `@Patron` | Pink | $50+/mo Patreon |
| `@Sponsor` | Cyan | $15+/mo |
| `@Supporter` | Blue | $5+/mo |
| `@Member` | Gray | Everyone else |

## Role Perks

### @Supporter ($5+/mo)
- Access to `#supporter-lounge`
- Access to `#early-access` channel
- Priority in `#installation-help`

### @Sponsor ($15+/mo)
- All Supporter perks
- Access to `#priority-support`
- Monthly dev update ping
- Feature request voting (reactions)

### @Patron ($50+/mo)
- All Sponsor perks
- Access to `#patron-chat`
- Quarterly roadmap call invite
- Special flair on GitHub/Reddit

## Bot Setup (Optional)

### Suggested Bots
- **Carl-bot** — Reaction roles, embeds
- **Ticket Tool** — Private support tickets
- **Patreon Bot** — Auto-role for patrons
- **GitHub Bot** — Repo updates in `#announcements`

### Patreon Integration
1. Link Patreon to Discord in Patreon settings
2. Bot auto-assigns roles based on tier
3. Supporters get instant access to channels
