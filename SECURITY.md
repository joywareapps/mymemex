# Security Policy

## Reporting a Vulnerability

We take the security of MyMemex seriously. If you believe you have found a security vulnerability, please **do not open a public issue**.

Instead, please report it to us privately via email:

**Email:** support@joywareapps.com

We will acknowledge your report within 48 hours and provide a timeline for resolution.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | ✅ Yes             |
| < 0.1.0 | ❌ No              |

## Our Security Commitment

- We will respond quickly to all valid vulnerability reports.
- We will provide a fix or mitigation as soon as possible.
- We will give you credit for the find (unless you prefer to remain anonymous).
