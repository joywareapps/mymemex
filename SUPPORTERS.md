# Support Development

If it saves you time or you believe in the project, consider supporting development!

- **Monthly or one-time** donations
- **Badges** on your GitHub profile
- 🔗 [github.com/sponsors/joywareapps](https://github.com/sponsors/joywareapps)

### Patreon
- **Monthly subscriptions** with tiers

- **Early access** to new features
- 🔗 [patreon.com/joywareapps](https://patreon.com/joywareapps)

### Ko-fi
- **One-time donations** (buy me a coffee)

- 🔗 [ko-fi.com/joywareapps](https://ko-fi.com/joywareapps)

### Benefit Details

**🎯 Priority Issue Triage**
Your GitHub issues get labeled and triaged within 48 hours. 
While I can't guarantee fixes, your bugs get looked at first.

**🚀 Early Access Releases**
Get beta/RC builds 1-2 weeks before public release. 
Help shape features before they ship.

**📧 Priority Support Email**
Direct email support with 72-hour response time. 
For debugging, configuration help, and usage questions.

**🗳️ Feature Request Voting**
Vote on which features get prioritized next. 
Monthly polls for upcoming milestones.

**📝 Monthly Dev Updates**
Patreon-exclusive posts with:
- What's being worked on
- Behind-the-scenes decisions
- Upcoming features preview
- Lessons learned

**📞 Roadmap Input Call**
Quarterly 30-minute video call to discuss:
- Your use cases and needs
- Feature priorities
- Product direction feedback

**🖼️ Logo on README**
Your name/company logo appears in the Supporters section of README.

**🔧 Custom Feature Request**
One custom feature or integration request per year, 
evaluated and potentially implemented.
